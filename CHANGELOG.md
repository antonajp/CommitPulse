# Changelog

All notable changes to the CommitPulse extension will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2026-03-10

### Added

- Initial release of CommitPulse VS Code extension
- Git commit extraction with PostgreSQL storage
- Jira and Linear issue tracker integration
- GitHub contributor sync
- Team assignment and analytics
- 11 interactive analytics dashboards (Sprint Velocity, Development Pipeline, Release Management, Code Review Velocity, Hot Spots, Knowledge Concentration, Ticket Lifecycle, Developer Focus, Team Coupling, Release Risk, Test Debt Predictor, Commit Hygiene, Architecture Drift)
- Pipeline scheduler with cron support
- TreeView panels for repositories, contributors, and pipeline runs
- SCC integration for code complexity metrics
- Architecture component classification
- Secure credential storage via VS Code SecretStorage
- Docker PostgreSQL with automatic migrations
